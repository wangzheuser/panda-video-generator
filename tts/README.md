# TTS Tool

Text-to-speech tool using Edge-TTS (Microsoft Edge Text-to-Speech).

## Setup

```bash
# Create virtual environment
python3 -m venv tts/venv

# Activate virtual environment
source tts/venv/bin/activate  # macOS/Linux
# or
tts\venv\Scripts\activate  # Windows

# Install dependencies
pip install -r tts/requirements.txt
```

## Usage

```bash
source tts/venv/bin/activate
python tts/tts.py output/tts/input.txt output/tts
```

The tool reads the subtitle file from `output/tts/input.txt` (generated by the spider's caption generator) and generates audio files.

### Parameters

- **Input file**: Path to the subtitle text file (default: `output/tts/input.txt`)
- **Output directory**: Directory where audio files will be saved (default: `output/tts`)

### Examples

```bash
# Use default input file
python tts/tts.py output/tts/input.txt output/tts

# Use custom input file
python tts/tts.py path/to/custom.txt output/tts

# Use custom output directory
python tts/tts.py output/tts/input.txt custom/output/dir
```

## Output

Generates in the output directory:
- `audio.mp3` - Merged audio file (speed adjusted to 1.2x)
- `audio.vtt` - VTT subtitle file with timestamps

Temporary sentence files are automatically deleted after merging.

## Workflow Integration

The TTS tool is designed to work seamlessly with the spider's caption generator:

1. Run the spider to extract content and generate captions: `pnpm spider <url>`
2. Caption file is automatically saved to `output/tts/input.txt`
3. Run TTS to generate audio: `python tts/tts.py output/tts/input.txt output/tts`
4. Use the generated `audio.mp3` and `audio.vtt` files in your Remotion video (files are automatically copied to `public/tts/` for Remotion access)
